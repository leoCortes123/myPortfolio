Efectos de Texto Expandido (10-15px)

1. Múltiples sombras extremas

Método más compatible con todos los navegadores

TEXTO GRUESO
.extreme-shadow {
    text-shadow:
        10px 0 0 black, -10px 0 0 black, 0 10px 0 black, 0 -10px 0 black,
        7px 7px 0 black, -7px -7px 0 black, 7px -7px 0 black, -7px 7px 0 black,
        10px 10px 0 black, -10px -10px 0 black, 10px -10px 0 black, -10px 10px 0 black;
}

2. Pseudo-elemento corregido

Usando text-stroke con posición absoluta

TEXTO ANCHO
.working-pseudo::before {
    content: attr(data-text);
    position: absolute;
    z-index: -1;
    color: transparent;
    -webkit-text-stroke: 15px black;
    text-stroke: 15px black;
}

3. SVG con expansión fuerte

<svg width="100%" height="120">
    <defs>
        <filter id="thickOutline" x="-20%" y="-20%" width="140%" height="140%">
            <feMorphology in="SourceAlpha" result="expanded" operator="dilate" radius="8"/>
            <feFlood flood-color="black" result="flood"/>
            <feComposite in="flood" in2="expanded" operator="in" result="outline"/>
            <feComposite in="outline" in2="SourceGraphic" operator="over"/>
        </filter>
    </defs>
    <text x="50%" y="70" font-size="60" font-weight="bold" fill="white"
          filter="url(#thickOutline)" text-anchor="middle">SVG GRUESO</text>
</svg>
